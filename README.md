# map

새로운 언어 플러터를 2일만에 배우고 하루만에 만들어보는 지도 기반 맛집 검색 앱 !

지도 움직이면 움직인 위치 기준으로 맛집 갱신하고, 줌 하면 줌에 맞춰서 식당 리스트를 갱신해요

리스트 메뉴에선 좌우 플릭으로 맵에서 보여주었던 식당 리스트를 상세하게 볼 수 있어요
